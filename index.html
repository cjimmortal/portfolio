Flood Monitoring System

Obejctives

To track an increase in water levels, a flood monitoring system is utilized. The system consists of sensors that are placed in the hardware of the system. The sensors are connected to the microcontroller to the system. These sensors are installed on places with the hardware device to measure water levels in real-time and transfer data continually to the central data system administration over various networks using API SMS, or Wi-Fi.
